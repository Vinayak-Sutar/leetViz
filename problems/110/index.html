A tree is balanced if EVERY node (not just the root) has left and right subtrees whose heights differ by at most 1.

3
Use DFS (post-order traversal): compute the height of left subtree first, then right, then compare them at the current node.

You can combine the height computation and balance check: if any subtree is unbalanced, propagate a special value (like -1) upward to short-circuit.

height(node): if node is null return 0.
leftH = height(left), rightH = height(right).
If leftH == -1 or rightH == -1 or |leftH - rightH| > 1, return -1.
Else return max(leftH, rightH) + 1.
