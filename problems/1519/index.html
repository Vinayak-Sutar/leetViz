Count nodes with the same label in each subtree via DFS

Nodes: 7
Labels: "abaedcd"

1
A subtree of node i includes node i itself and all its descendants. Think about how DFS naturally visits entire subtrees.
🔒 Click to reveal
2
For each node, you need to know the frequency of each character in its subtree. A frequency counter (array of 26) is perfect for this.
🔒 Click to reveal
3
Use post-order DFS: first recurse into all children, then merge their frequency counters into the parent's counter.
🔒 Click to reveal
4
After merging children, add +1 for the current node's own label. The answer for node i is count[labels[i]].
🔒 Click to reveal
5
Build adjacency list from edges. DFS from root 0 with a visited set. At each node: count = [0]*26, recurse children, merge counts, count[label] += 1, set ans[node] = count[label], return count.
