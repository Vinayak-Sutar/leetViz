Imagine writing on lined paper. You go down one line at a time, then back up — like a zigzag or a "V" shape repeating.

2
The key pattern: characters go down rows 0→(numRows-1), then diagonally up rows (numRows-2)→1, then repeat.
🔒 Click to reveal
3
Use an array of strings — one per row. Track the current row and direction. Append each character to the appropriate row string.
🔒 Click to reveal
4
Direction flips when you hit row 0 (start going down) or row numRows-1 (start going up). Use a boolean or +1/-1 step variable.
🔒 Click to reveal
5
rows = [""] * numRows, curRow = 0, step = -1.
For each char: rows[curRow] += char. If curRow == 0 or curRow == numRows-1: step *= -1. Then curRow += step.
Answer = "".join(rows).
