Think of it as distributing turnedOn LEDs between hours (0–3 LEDs) and minutes (0–6 LEDs). For each split (h, m) where h + m = turnedOn, find valid combinations.

A simpler approach: iterate through all hours (0–11) and minutes (0–59). For each combo, count the total set bits. If it equals turnedOn, it's a valid time.

for h in 0..11: for m in 0..59: if popcount(h) + popcount(m) == turnedOn: add formatted time. Complexity is O(1) since the loop is bounded by constants (12 × 60 = 720).
